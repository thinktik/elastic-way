# eagle
eagle: 鹰

身份认证系统

**功能清单**：
- [ ] 用户注册，用户登陆
- [ ] 用户权限管理
- [ ] Oauth2
- [ ] 集成第三方身份认证系统
  - [ ] QQ
  - [ ] wechat
  - [ ] github
  - [ ] microsoft
  - [ ] google
  - [ ] Amazon
